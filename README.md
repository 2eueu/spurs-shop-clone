# Tottenham Hotspur Shop

### **프로젝트 개요**
토트넘 홋스퍼의 굿즈 웹사이트를 모방한 프로젝트입니다.  
사용자가 키트, 나이키 제품을 검색하고 구매하며, 특정 선수 마킹까지 선택할 수 있는 사이트를 목표로 제작했습니다.
